LOOPS

Loops are used in JavaScript to perform repeated tasks based on a condition. Conditions typically return true or false. A loop will continue running until the defined condition returns false. For example, if you want to show a message 100 times, then you can use a loop. It's just a simple example; you can achieve much more with loops.

There are three different types of loops such as:

WHILE LOOP



FOR LOOP



DO WHILE LOOP


LOOPS CONTROLS

The Java for loop is a control flow statement that iterates a part of the programs multiple times. The Java while loop is a control flow statement that executes a part of the programs repeatedly on the basis of given boolean condition.

There are two types of loop control that are:

BREAK LOOP



CONTINUE LOOP
